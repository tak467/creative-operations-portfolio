# Creative Operations Portfolio

映像制作会社向けに内製した、
業務基幹システム・制作支援ツール群のポートフォリオです。

## 背景
少人数で複数拠点の制作業務を回す中で、
案件管理・人材配置・制作進行・売上管理の属人化が課題になっていました。

それらを改善するため、
AI駆動開発を活用しながら、
業務システムを内製・運用しています。

## 主な実績
- 受注〜人選〜売上〜請求管理システム
- 人選タイムライン可視化
- 売上ダッシュボード
- メディア管理
- 印刷スケジュール生成

## 技術スタック
- Python
- Flask
- SQLite
- JavaScript
- GitHub
- VPS / systemd
